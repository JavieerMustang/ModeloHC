# Humidity Cell Pilot Laboratory (ModeloHC) 

Welcome to the **ModeloHC** repository. This project features an interactive 3D visualization and a computational simulation of a custom-engineered Humidity Cell Pilot Laboratory. 

This digital framework was developed to evaluate the kinetics of sulfide oxidation and the generation of acid rock drainage (ARD) in mine tailings, with a special focus on semi-arid climates and the precipitation of efflorescent salts.

##  Scientific Background

The management of mine waste and the prediction of Acid Rock Drainage (ARD) are critical environmental challenges. While the standard **ASTM D5744-18** humidity cell test is globally accepted, its rigid application often fails to accurately reproduce weathering rates in hyper-arid environments, such as northern Chile. 

This project moves beyond standard commercial equipment by proposing a **dual-line system**. It is designed to induce and study the fractional precipitation of complex efflorescent salts (e.g., copiapite, halotrichite), which temporarily sequester metals and acidity, leading to sudden contamination events ("store and release" effect) that standard tests often miss.

##  Key Features

* **Hydrodynamic Validation:** 3D spatial layout demonstrating the integration of dry/humid air lines and leaching systems.
* **Geometallurgical Focus:** Aligned with modern approaches that prioritize microtexture and mineral liberation over bulk chemistry.
* **Open-Source Framework:** Built using web technologies (HTML, CSS, JS) to make complex geological protocols accessible and reproducible.
